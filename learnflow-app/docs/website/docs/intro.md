# LearnFlow Introduction

LearnFlow is an innovative interactive Python learning platform that combines AI-powered tutoring with hands-on coding experiences. Our platform helps students master Python programming concepts through guided lessons, real-time feedback, and personalized learning paths.

## Key Features

- **Interactive Code Editor**: Built with Monaco Editor for a rich coding experience
- **AI-Powered Tutoring**: Intelligent assistance for learning Python concepts
- **Progress Tracking**: Detailed analytics on student progress and mastery
- **Gamification**: Streak tracking and achievement systems to maintain engagement
- **Real-time Collaboration**: Live chat with AI tutors and concept explanations
- **Adaptive Learning**: Personalized content based on individual progress

## Architecture Overview

LearnFlow is built using a modern microservices architecture:

- **Frontend**: Next.js application with TypeScript and Tailwind CSS
- **Backend**: FastAPI services for business logic and data processing
- **Triage Agent**: AI-powered routing system for student queries
- **Concepts Agent**: Specialized AI agent for explaining Python concepts
- **Database**: PostgreSQL for persistent data storage
- **Message Queue**: Apache Kafka for asynchronous processing
- **Authentication**: Better Auth for secure user management

## Getting Started

To start using LearnFlow, navigate to the student dashboard to begin your Python learning journey, or access the teacher dashboard to manage classes and track student progress.
