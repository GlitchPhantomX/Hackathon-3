# LearnFlow App

A comprehensive learning platform that leverages AI for personalized education and skill development.

## Overview

LearnFlow is an intelligent learning platform that uses AI to provide personalized learning experiences. The platform includes features for content discovery, skill assessment, and adaptive learning paths.

## Architecture

The application follows a microservices architecture with the following components:

- **Frontend**: User interface built with modern web technologies
- **Triage Service**: Handles initial user assessment and learning path recommendation
- **Concepts Service**: Manages learning concepts and knowledge graphs
- **Debug Service**: Provides debugging assistance for programming exercises
- **Exercise Service**: Handles interactive coding exercises and assessments

## Prerequisites

- Node.js 18+ for frontend development
- Python 3.9+ for backend services
- Docker and Docker Compose
- Kubernetes cluster for deployment
- Dapr for distributed application runtime

## Setup Instructions

1. Clone the repository
2. Set up the backend services
3. Configure Dapr and infrastructure components
4. Build and run the frontend application
5. Access the application through the provided endpoints
