# Kafka K8s Setup Skill

## Description
This skill automates the setup and configuration of Apache Kafka clusters in Kubernetes environments.

## Capabilities
- Deploy Kafka brokers and Zookeeper
- Configure topic replication and partitions
- Set up monitoring and logging
- Configure security and authentication
- Scale cluster resources

## Input Parameters
- Number of Kafka brokers
- Storage configuration
- Network settings
- Security requirements
- Resource limits and requests

## Output
- Running Kafka cluster in Kubernetes
- Configured topics and partitions
- Monitoring dashboards
- Connection details and configuration files
