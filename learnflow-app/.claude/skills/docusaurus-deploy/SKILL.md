# Docusaurus Deploy Skill

## Description
This skill automates the deployment of Docusaurus documentation sites with optimized configuration and hosting.

## Capabilities
- Build and optimize Docusaurus sites
- Configure CDN and caching
- Set up custom domain and SSL
- Implement versioned documentation
- Configure search and analytics

## Input Parameters
- Docusaurus site source code
- Site configuration and metadata
- Domain and SSL certificate information
- Analytics and search configuration
- Versioning requirements

## Output
- Deployed Docusaurus site in Kubernetes
- CDN and caching configuration
- SSL certificate setup
- Search and analytics integration
- Versioned documentation structure
