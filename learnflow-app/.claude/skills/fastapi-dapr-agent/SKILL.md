# FastAPI Dapr Agent Skill

## Description
This skill creates FastAPI services with Dapr (Distributed Application Runtime) integration for building cloud-native applications.

## Capabilities
- Generate FastAPI service templates with Dapr integration
- Configure Dapr components (state stores, pub/sub, secrets)
- Set up service-to-service communication
- Configure observability and tracing
- Deploy to Kubernetes with Dapr sidecar injection

## Input Parameters
- Service name and description
- API endpoints and models
- Dapr component configurations
- State store requirements
- Pub/sub topic definitions

## Output
- FastAPI service with Dapr integration
- Dapr component configuration files
- Kubernetes deployment manifests
- API documentation and examples
