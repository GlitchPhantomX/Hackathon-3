# Agents Markdown Generator Reference

## Architecture
The agents-md-gen skill follows a modular architecture that allows for flexible generation of agent documentation.

## Components
- Parser: Parses agent configuration and architecture
- Template Engine: Renders markdown templates
- Diagram Generator: Creates architecture diagrams
- Validator: Ensures documentation quality

## Templates
The skill uses predefined templates for different types of documentation:
- Agent overview
- API documentation
- Configuration guide
- Deployment instructions

## Integration Points
- Supports various agent frameworks
- Integrates with documentation generators
- Compatible with version control systems
