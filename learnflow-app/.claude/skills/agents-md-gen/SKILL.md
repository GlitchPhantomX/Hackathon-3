# Agents Markdown Generator Skill

## Description
This skill generates comprehensive markdown documentation for AI agents, including architecture diagrams, API specifications, and usage examples.

## Capabilities
- Generate agent architecture documentation
- Create API endpoint documentation
- Document configuration parameters
- Provide usage examples and best practices

## Input Parameters
- Agent name and description
- Architecture details
- API specifications
- Configuration requirements

## Output
- Well-formatted markdown documentation
- Architecture diagrams
- Code examples
- Best practice recommendations
