# Next.js K8s Deploy Skill

## Description
This skill automates the deployment of Next.js applications to Kubernetes environments with optimal configuration.

## Capabilities
- Build and containerize Next.js applications
- Configure ingress for external access
- Set up CDN and caching strategies
- Configure SSL termination
- Implement blue-green deployment

## Input Parameters
- Next.js application source code
- Environment variables and configuration
- Domain and SSL certificate information
- Resource requirements
- Scaling policies

## Output
- Deployed Next.js application in Kubernetes
- Configured ingress and load balancing
- CDN and caching setup
- SSL certificate configuration
- Health checks and monitoring
