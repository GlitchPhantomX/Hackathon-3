# Skills Library

A collection of reusable AI skills for various development tasks and infrastructure management.

## Overview

This repository contains a library of reusable AI skills that can be leveraged for different projects. Each skill is designed to be self-contained and easily integrable into different workflows.

## Skills Included

- **agents-md-gen**: Generate comprehensive documentation for AI agents
- **kafka-k8s-setup**: Set up Kafka clusters in Kubernetes environments
- **postgres-k8s-setup**: Deploy PostgreSQL databases in Kubernetes
- **fastapi-dapr-agent**: Create FastAPI services with Dapr integration
- **mcp-code-execution**: Execute code in managed container platforms
- **nextjs-k8s-deploy**: Deploy Next.js applications to Kubernetes
- **docusaurus-deploy**: Deploy documentation sites with Dapr integration

## Usage

Each skill includes detailed documentation in its respective folder. Refer to the individual skill documentation for specific usage instructions.
