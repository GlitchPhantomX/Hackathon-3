# Agents MD Generator

## When to Use
Use this skill to automatically generate an AGENTS.md file that documents all AI agent components in a repository.

## Instructions
1. Run the `generate_agents_md.py` script from the scripts/ folder
2. The script will analyze the repository structure and identify AI agent components
3. A properly formatted AGENTS.md file will be created in the repository root

## Validation Checklist
- [ ] Repository structure analyzed successfully
- [ ] Agent files identified and documented
- [ ] AGENTS.md file created with proper format
- [ ] All agent components are documented
